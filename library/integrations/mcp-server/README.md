## Overview

This MCP server allows AI assistants to:
- Execute 880+ automation scripts (0000-9999 number-based system)
- Run infrastructure deployments and VM management
- Execute orchestrated playbooks for complex workflows
- Manage configurations from config.psd1 manifest
- Run tests and quality checks across aithercore domains
- Generate documentation and project reports
- Access GitHub workflow status
- Use guided prompts for multi-step operations

For complete documentation, see the full README in the integrations/mcp-server directory.
