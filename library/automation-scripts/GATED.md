Some script categories are not included in the public release:

- `00-bootstrap` — Full AitherOS bootstrap (enterprise license)
- `01-infrastructure` — Infrastructure provisioning
- `08-aitheros` — AitherOS lifecycle management
- `20-build` — Build pipeline
- `26-roboflow` — Roboflow 3D asset pipeline (project-specific)
- `27-aither-integrations` — AitherOS-internal product integrations
- `30-deploy` — Deployment playbooks (enterprise license)
- `31-remote` — Remote deployment
- `40-lifecycle` — Service lifecycle management
- `50-ai-setup` — AI infrastructure setup
- `60-security` — Security infrastructure
- `70-github` — GitHub org/runner bootstrap (org-specific)

These are available with an AitherOS Enterprise license.
Contact hello@aitherium.com for details.
